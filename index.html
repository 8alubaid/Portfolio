Dockerized Message Board Web Application

As part of a collaborative team project, I helped build a full-stack forum web application using Node.js, PostgreSQL, and Docker. I developed Express.js backend routes handling authentication, board and post creation, and protected page access, while integrating PostgreSQL with relational queries to manage users, discussion boards, posts, and comments. Node.js PostgreSQL Docker
03
Golden Arduino Printed Circuit Board Design Project

For the Golden Arduino PCB Design Project, I designed a custom Arduino-compatible PCB in Altium Designer with USB programming support and Uno R3 shield compatibility. I optimized the PCB layout to minimize signal noise through strategic use of decoupling capacitors, shorter traces, and improved component placement, while integrating test points, indicator LEDs, power regulation, and USB-to-UART communication circuitry. To validate the design, I performed oscilloscope-based signal integrity and power noise analysis, benchmarking the board's performance against a commercial Arduino.

Altium Designer PCB Design
